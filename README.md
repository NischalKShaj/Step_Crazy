# StepCrazy E-commerce Platform

## Overview

stepcrazy is a comprehensive e-commerce platform developed and deployed to provide a seamless online shopping experience for shoe enthusiasts. The platform incorporates various features, including product view, zoom, cart/wishlist management, diverse payment methods, order details, coupon application, and PDF invoices. Additionally, it includes an admin dashboard for graphical sales representation, user management, and product/category/coupon editing.

## Features

- **Product View and Zoom:**
  - Interactive product views with zoom functionality to enhance the user experience.

- **Cart/Wishlist Management:**
  - Effortless cart and wishlist management for users to track and organize their selected items.

- **Diverse Payment Methods:**
  - Multiple payment methods supported, including Cash on Delivery (COD), Razorpay, and wallet payments.

- **Order Details:**
  - Detailed order information to keep users informed about their purchases.

- **Coupon Application:**
  - Ability for users to apply coupons for discounts during checkout.

- **PDF Invoices:**
  - Automatic generation of PDF invoices for user orders.

- **Admin Dashboard:**
  - Graphical representation of sales data for quick insights.
  - User management capabilities.
  - Editing options for products, categories, and coupons.

## Technology Stack

- **Backend:**
  - Node.js and Express.js for scalable backend development.
  - MongoDB for efficient product data management.
  - RestAPI for seamless integration of server-side data.

- **Frontend:**
  - EJS for dynamic and interactive frontend design.

- **Additional Libraries:**
  - Nodemailer for email notifications.
  - PDFKit for PDF invoice generation.
  - Multer for handling file uploads.

- **Web Server:**
  - Nginx for server configuration and optimization.

- **Cloud Hosting:**
  - AWS for reliable and scalable cloud hosting.
